# Mirko - Italian Software Developer 🇮🇹

Welcome to the official repository of Mirko's personal website and portfolio, an Italian developer passionate about technology and programming.

## Project Description

This repository contains the static export of the website [https://devmirko.info](https://devmirko.info), a personal portfolio designed to showcase Mirko's skills, projects, educational background, and growth journey in software development.

The website consists of the following main sections:

- **Info**: details about programming skills, work experience, education certificates, and recent projects.
- **About**: a personal narrative describing Mirko’s life path, from the first encounters with technology to university studies and professional aspirations.
- **Projects**: a detailed list of current and past projects Mirko has worked on, focusing on Roblox game development, Discord bots, and web applications.

## Technologies Used

- Modern web framework (Next.js and React)
- Languages: JavaScript, TypeScript, Lua (for Roblox), HTML5, CSS3
- Node.js backend for Discord bots and custom integrations
- Libraries and APIs such as discord.js and MySQL

## Usage

This repository contains a fully exported static site ready to be deployed on any static hosting platform (e.g., GitHub Pages, Vercel, Netlify).

To view the site, simply open the `index.html` file in a web browser.
